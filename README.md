# 🎓 Student Record System (C Language)

A simple Student Record Management System built using C.

## 📌 Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Prevent Duplicate IDs
- File Handling using record.txt

## 🛠 Tech Used
- C Programming
- GCC Compiler (MinGW)
- Git & GitHub

## ▶️ How to Run

```bash
gcc student.c -o student
./student
